Nagpur Metro – Official Website

Responsible for maintaining and enhancing the Nagpur Metro Rail official website. The platform is developed in ASP.NET Web Forms (VB.NET) with SQL Server as backend, supporting English & Marathi languages to serve a diverse audience.

  • Maintained & developed new commuter-facing features
  • Implemented multilingual support (English/Marathi)
  • Optimized performance & ensured smooth uptime
  • Managed SQL Server database with secure data handling
  • Enhanced SEO with meta tags, structured data & sitemaps
  • Improved Accessibility with alt text, ARIA labels & keyboard navigation
  • Delivered responsive design across devices & browsers

Impact: Improved digital presence of Nagpur Metro, ensured seamless access for thousands of daily users, boosted search visibility, and delivered an inclusive experience.
