# Swastik Dharpure - Portfolio Website

## Description
This is the personal portfolio website of **Swastik Dharpure**, a passionate .NET Developer and Web Developer from Nagpur.  
The website showcases projects, skills, and contact information, built with modern web technologies.

## Features
- Responsive and mobile-friendly design
- Projects showcase with images
- Skills section
- Contact form (static, can integrate SMTP/email)
- Smooth scrolling and animations
- Built using HTML5, CSS3, JavaScript, Bootstrap, and vendor libraries like AOS, Glightbox, Swiper

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Vendor Libraries:** AOS, Glightbox, Swiper, Bootstrap Icons
- **Other Tools:** Google Fonts, Favicon for multiple devices
